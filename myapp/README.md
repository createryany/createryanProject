# 所用框架 Ant Design Pro

## Environment Prepare

安装 `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## 项目脚本

 使用 Ant Design Pro 提供的脚本来快速启动和构建 web 项目、代码检查和测试。

### 启动项目
```bash
npm start
```

### 构建项目
```bash
npm run build
```

### 检查代码
```bash
npm run lint
```

您还可以使用脚本自动修复一些 lint 错误：
```bash
npm run lint:fix
```

## 测试代码

```bash
npm test
```
